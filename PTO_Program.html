PTO Program

PTO V1.0

A simple program that calculates the PTO (Paid Time Off) for 1yr.

  1. START
  2. SET IN paychecksPerYear=26
  3. SET IN hrsPerPaycheck=5
  4. DISPLAY "This is a simple program that outputs how many hours you earn a year for PTO(Paid Time Off)."
  5. SET IN totalHoursPerYear = hrsPerPayCheck * paychecksPerYear
  6. DISPLAY "You earn totalHoursPerYear hours per year."
  7. END
